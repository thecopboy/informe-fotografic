## [1.0.0] - 2025-07-14

### Afegit
- Aplicació completa per informes fotogràfics
- Sistema d'autenticació amb JWT
- Gestió d'usuaris i perfils
- Pujada i gestió d'imatges
- Generació de PDFs professionals
- Interfície responsiva (desktop i mòbil)
- Sistema de notificacions
- Importació/exportació d'informes
- Web Components per gestió d'imatges
- Sistema de logging centralitzat
- Monitorització i mètriques
- Tests automatitzats
- Documentació completa

### Característiques Tècniques
- Arquitectura modular amb ES6+ modules
- Patrons de disseny (DI, Observer, Factory, Command)
- Web Components amb Shadow DOM
- Sistema d'estat reactiu
- Drag & Drop centralitzat
- Base de dades SQLite
- API RESTful amb Express.js
- Seguretat amb Helmet i rate limiting
- Optimització de rendiment

### Desenvolupament
- Assistència d'IA en el desenvolupament
- Codi net i mantenible
- Documentació exhaustiva
- Tests de cobertura
- Configuració flexible
